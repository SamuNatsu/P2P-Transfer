# Frontend

## Build

1. Install dependencies

```bash
# You can use other package managers
pnpm install
```

2. Execute build script

```bash
pnpm run build
```

3. Distributions are in `dist` directory

## Development

1. Install dependencies

```bash
# You can use other package managers
pnpm install
```

2. Start dev server

```bash
pnpm run dev
```
